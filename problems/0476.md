# [Problem 476: Number Complement](https://leetcode.com/problems/number-complement/)

## Solution

```py
# Time complexity: O(1)
# Space complexity: O(1)
class Solution:
    def findComplement(self, num: int) -> int:
        return (~num) & ~(-1 << int(log(num, 2)))
```
