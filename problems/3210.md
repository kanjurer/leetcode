# [Problem 3210: Find the Encrypted String](https://leetcode.com/problems/find-the-encrypted-string/)

## Solution

```py
# Time complexity: O(n)
# Space complexity: O(1)
class Solution:
    def getEncryptedString(self, s: str, k: int) -> str:
        k = k % len(s)
        return s[k:] + s[:k]
```
