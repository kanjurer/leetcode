# [Problem 3174: Clear Digits](https://leetcode.com/problems/clear-digits/)

## Solution

```py
# Time complexity: O(n)
# Space complexity: O(n)
class Solution:
    def clearDigits(self, s: str) -> str:
        stack = []

        for c in s:
            if c not in "0123456789":
                stack.append(c)
            else:
                stack.pop()

        return "".join(stack)
```
